# Hi, I'm Zachary 👋

I build **game tooling and AI agents** — from a 40-mod Minecraft Fabric library to
an offline computer-vision research sandbox. I like small, well-factored systems
with clean interfaces, real test coverage, and honest documentation.

- 🔭 **Focus:** client-side game mods (Java / Fabric), applied AI (Python / CV /
  behavior trees), and developer-friendly architecture.
- 🧩 **Style:** decoupled layers, swappable backends, safety/fair-play by default,
  and READMEs that actually explain the design.
- 🌱 **Currently exploring:** agent architectures and perception pipelines.
